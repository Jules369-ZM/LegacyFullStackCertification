# Front End Development Libraries Projects

This directory contains implementations of the Front End Development Libraries certification projects from FreeCodeCamp. All projects are built using React with JSX and are fully functional.

## Projects Implemented

### 1. Random Quote Machine
- **Directory**: `random-quote-machine/`
- **Description**: A random quote generator with inspirational quotes from famous people
- **Features**:
  - Random quote generation
  - Tweet functionality
  - Color-changing background
  - Responsive design

### 2. Markdown Previewer
- **Directory**: `markdown-previewer/`
- **Description**: A live markdown editor and previewer
- **Features**:
  - Real-time markdown rendering
  - Split-pane layout (editor + preview)
  - Support for all markdown elements
  - Responsive design

### 3. Drum Machine
- **Directory**: `drum-machine/`
- **Description**: An interactive drum machine with playable drum pads
- **Features**:
  - 9 drum pads with different sounds
  - Click or keyboard controls (Q, W, E, A, S, D, Z, X, C)
  - Power on/off functionality
  - Volume control
  - Visual feedback

### 4. JavaScript Calculator
- **Directory**: `javascript-calculator/`
- **Description**: A fully functional calculator with basic arithmetic operations
- **Features**:
  - Addition, subtraction, multiplication, division
  - Decimal point support
  - Clear and equals functionality
  - Proper order of operations
  - Formula display

### 5. 25 + 5 Clock (Pomodoro Timer)
- **Directory**: `25-plus-5-clock/`
- **Description**: A Pomodoro technique timer for productivity
- **Features**:
  - Adjustable session and break lengths
  - Start/pause/reset controls
  - Audio alerts when timer reaches zero
  - Automatic switching between sessions and breaks
  - Circular timer display

## Technologies Used

- **React**: All projects use React for component-based UI
- **JSX**: JavaScript XML for component rendering
- **CSS**: Responsive styling with media queries
- **HTML5 Audio**: For sound effects in Drum Machine and Pomodoro Timer
- **CDN Libraries**: React, Babel, Marked.js, Font Awesome

## Running the Projects

Each project is self-contained and can be run by opening the `index.html` file in a web browser. No build process or server setup is required.

## FreeCodeCamp Certification

These implementations fulfill the requirements for the Front End Development Libraries certification from FreeCodeCamp.
