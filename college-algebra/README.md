# College Algebra with Python

This directory contains implementations of the College Algebra with Python certification projects from FreeCodeCamp. All projects focus on mathematical computations, data visualization, financial calculations, and interactive educational tools using Python.

## Projects Implemented

### 1. Multi-Function Calculator (`multi_function_calculator/`)
- **Description**: Comprehensive calculator with arithmetic, scientific, statistical, and algebraic functions
- **Features**:
  - Basic arithmetic operations (+, -, *, /, ^, %)
  - Scientific functions (sin, cos, tan, log, exp, sqrt)
  - Statistical calculations (mean, median, mode, std dev)
  - Quadratic equation solver
  - Memory functions and calculation history
  - Safe expression evaluation with restricted globals

### 2. Graphing Calculator (`graphing_calculator/`)
- **Description**: Interactive mathematical function plotting and analysis tool
- **Libraries**: NumPy, Matplotlib, SciPy
- **Features**:
  - Plot multiple mathematical functions simultaneously
  - Root finding using numerical methods (bisection)
  - Extrema finding (local maxima and minima)
  - Definite integral calculation (numerical integration)
  - Function evaluation at specific points
  - Interactive command-line interface

### 3. Three Math Games (`math_games/`)
- **Description**: Educational games to practice mathematical concepts
- **Game Types**: Arithmetic, Algebra, Geometry, Trigonometry, Mixed Review
- **Features**:
  - Multiple difficulty levels (easy, medium, hard)
  - Different mathematical topics coverage
  - Score tracking and performance feedback
  - Timed gameplay with statistics
  - Interactive question generation

### 4. Financial Calculator (`financial_calculator/`)
- **Description**: Comprehensive financial calculation tool
- **Calculations**: Loans, Investments, Savings, Retirement, Taxes, Break-even
- **Features**:
  - Loan/mortgage payment calculations with amortization
  - Investment growth with compound interest
  - Savings goal planning
  - Retirement projections with inflation adjustment
  - Tax impact analysis with progressive brackets
  - Break-even analysis for business decisions

### 5. Data Graph Explorer (`data_graph_explorer/`)
- **Description**: Interactive data visualization and analysis tool
- **Libraries**: pandas, NumPy, Matplotlib, Seaborn, scikit-learn
- **Features**:
  - Multiple chart types (histograms, boxplots, scatter plots, bar charts, etc.)
  - Correlation analysis and heatmaps
  - Principal Component Analysis (PCA)
  - Statistical summaries and data exploration
  - Support for CSV, Excel, and JSON data formats

## Technologies Used

- **NumPy**: Numerical computing and array operations
- **Matplotlib**: 2D plotting and data visualization
- **pandas**: Data manipulation and analysis
- **SciPy**: Scientific computing and advanced mathematical functions
- **scikit-learn**: Machine learning algorithms for PCA and preprocessing
- **Seaborn**: Statistical data visualization

## Prerequisites

- Python 3.x installed
- Required libraries: numpy, matplotlib, pandas, scipy, scikit-learn, seaborn
- Optional: jupyter for interactive data exploration

## Mathematical Concepts Covered

### Algebra
- **Linear Equations**: Solving ax + b = c
- **Quadratic Equations**: Root finding and factorization
- **Polynomial Operations**: Evaluation and manipulation
- **Systems of Equations**: Basic solution methods

### Calculus
- **Limits**: Numerical approximation approaches
- **Derivatives**: Rate of change calculations
- **Integrals**: Definite integral calculation using numerical methods
- **Extrema**: Finding local maxima and minima

### Statistics
- **Descriptive Statistics**: Mean, median, mode, standard deviation
- **Data Visualization**: Histograms, boxplots, scatter plots
- **Correlation Analysis**: Relationship between variables
- **Distribution Analysis**: Normal distributions and outliers

### Financial Mathematics
- **Compound Interest**: Future value calculations
- **Loan Amortization**: Payment schedules and interest calculations
- **Annuities**: Present and future value of regular payments
- **Tax Calculations**: Progressive tax systems
- **Break-even Analysis**: Cost-volume-profit relationships
