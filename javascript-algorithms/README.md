# JavaScript Algorithms and Data Structures Projects

This directory contains implementations of the JavaScript Algorithms and Data Structures certification projects from FreeCodeCamp.

## Projects Implemented

### 1. Palindrome Checker
- **File**: `palindrome-checker.js`
- **Function**: `palindrome(str)`
- **Description**: Checks if a given string is a palindrome, ignoring punctuation, case, and spacing.

### 2. Roman Numeral Converter
- **File**: `roman-numeral-converter.js`
- **Function**: `convertToRoman(num)`
- **Description**: Converts a decimal number to a Roman numeral.

### 3. Caesars Cipher (ROT13)
- **File**: `caesars-cipher.js`
- **Function**: `rot13(str)`
- **Description**: Implements a ROT13 cipher, shifting letters by 13 positions.

### 4. Telephone Number Validator
- **File**: `telephone-number-validator.js`
- **Function**: `telephoneCheck(str)`
- **Description**: Validates US telephone numbers in various formats.

### 5. Cash Register
- **File**: `cash-register.js`
- **Function**: `checkCashRegister(price, cash, cid)`
- **Description**: Calculates change and handles cash register operations.

## Running the Code

To test any algorithm, use Node.js:

```bash
cd javascript-algorithms
node palindrome-checker.js
node roman-numeral-converter.js
node caesars-cipher.js
node telephone-number-validator.js
node cash-register.js
```

## FreeCodeCamp Certification

These implementations fulfill the requirements for the JavaScript Algorithms and Data Structures certification from FreeCodeCamp.
