# Legacy JavaScript Algorithms and Data Structures Projects

This directory contains additional FreeCodeCamp JavaScript algorithms and data structures projects that demonstrate advanced programming concepts and problem-solving skills.

## Projects Implemented

### 1. Palindrome Checker (`palindrome-checker/`)
- **Description**: Function to check if a string is a palindrome
- **Features**: Ignores case, punctuation, and non-alphanumeric characters
- **Algorithm**: Two-pointer approach for efficient palindrome checking

### 2. Roman Numeral Converter (`roman-numeral-converter/`)
- **Description**: Converts decimal numbers to Roman numerals
- **Features**: Handles numbers from 1 to 3999
- **Algorithm**: Greedy algorithm using Roman numeral mapping

### 3. Telephone Number Validator (`telephone-number-validator/`)
- **Description**: Validates US telephone number formats
- **Features**: Supports multiple valid formats (area code optional, parentheses, dashes)
- **Algorithm**: Regular expressions for pattern matching

### 4. Cash Register (`cash-register/`)
- **Description**: Calculates change and manages cash register operations
- **Features**: Handles insufficient funds, exact change, and multiple denominations
- **Algorithm**: Greedy algorithm for optimal change calculation

### 5. RPG Creature Search App (`rpg-creature-search/`)
- **Description**: Interactive web app for searching and filtering RPG creatures
- **Features**:
  - Search by name, type, or description
  - Filter by creature type, challenge rating, and size
  - Detailed creature cards with stats
  - Modal popup with full creature information
  - Responsive design for all devices
  - Real-time filtering and search
