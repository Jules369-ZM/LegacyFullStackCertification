# Legacy Responsive Web Design Certification V8

This repository contains the projects completed for the FreeCodeCamp Legacy Responsive Web Design Certification.

## About

The Responsive Web Design Certification demonstrates proficiency in building responsive websites using HTML and CSS. This certification is part of the requirements for the Legacy Full Stack Certification.

## Projects

1. **Tribute Page** (`tribute-page/`) - A responsive tribute page dedicated to someone you admire, featuring images, text, and links.

2. **Survey Form** (`survey-form/`) - A responsive survey form with various input types and validation.

3. **Product Landing Page** (`product-landing-page/`) - A responsive landing page for a product with features, pricing, and call-to-action sections.

4. **Technical Documentation Page** (`technical-documentation-page/`) - A responsive technical documentation page with navigation and content sections.

5. **Personal Portfolio Webpage** (`personal-portfolio-webpage/`) - A responsive personal portfolio showcasing projects and contact information.

## Technologies Used

- HTML5
- CSS3 (including Flexbox and Grid for layouts)
- Responsive design principles
- Mobile-first approach

## How to View Projects

Each project is contained in its own folder. To view a project:

1. Navigate to the project folder
2. Open `index.html` in a web browser
3. Resize the browser window to test responsiveness

## Certification Requirements Met

This repository fulfills the Legacy Responsive Web Design Certification requirements as part of the path to the Legacy Full Stack Certification.
