# Data Visualization Projects

This directory contains implementations of the Data Visualization certification projects from FreeCodeCamp. All projects use D3.js for creating interactive data visualizations.

## Projects Implemented

### 1. Bar Chart (`bar-chart/`)
- **Description**: Visualizes US GDP data from 1947-2015 using a bar chart
- **Data Source**: FreeCodeCamp GDP dataset
- **Features**:
  - Interactive bars with hover tooltips
  - Time-based X-axis (years)
  - GDP values on Y-axis
  - Responsive design
  - Professional styling

### 2. Scatterplot Graph (`scatterplot-graph/`)
- **Description**: Scatter plot showing doping allegations in professional cycling
- **Features**: Coming soon...

### 3. Heat Map (`heat-map/`)
- **Description**: Heat map visualization of global temperature data
- **Features**: Coming soon...

### 4. Choropleth Map (`choropleth-map/`)
- **Description**: Choropleth map showing US education attainment by county
- **Features**: Coming soon...

### 5. Treemap Diagram (`treemap-diagram/`)
- **Description**: Treemap showing video game sales data
- **Features**: Coming soon...

## Technologies Used

- **D3.js**: Primary library for data visualization
- **HTML5 & CSS3**: Structure and styling
- **JavaScript (ES6+)**: Data manipulation and interactivity
- **FreeCodeCamp APIs**: Data sources for all visualizations

## Running the Projects

Each project is self-contained and can be run by opening the `index.html` file in a web browser. The projects fetch data from FreeCodeCamp's APIs and render interactive visualizations.

## FreeCodeCamp Certification

These implementations fulfill the requirements for the Data Visualization certification from FreeCodeCamp.
