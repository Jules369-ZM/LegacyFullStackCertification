FreeCodeCamp Back End Development and APIs

Timestamp Microservice

Get timestamp information in Unix and UTC formats.

Examples:

  • /api/timestamp
  • /api/timestamp/2023-01-01
  • /api/timestamp/1672531200000

Request Header Parser Microservice

Get information about your request headers.

Example:

  • /api/whoami

URL Shortener Microservice

Create short URLs and redirect to original URLs.

Examples:

  • POST /api/shorturl with body {"url": "https://example.com"}
  • /api/shorturl/1

Exercise Tracker

Track your exercise activities (requires MongoDB).

Examples:

  • POST /api/users with body {"username": "testuser"}
  • GET /api/users
  • POST /api/users/{userId}/exercises
  • GET /api/users/{userId}/logs

File Metadata Microservice

Upload files and get metadata information.

Example:

  • POST /api/fileanalyse with file upload

Built with Node.js, Express.js, and MongoDB
